# Simulation Code

## Scripts

- run_mujoco_simulation.py: reference test.
- run_mujoco_simulation2.py: runs the simulation with a server.
- run_mujoco_simulation2matplot.py: runs the simulation with matplotlib in case the server does not work.

## PID Tuning

To change PID values, edit Kp, Ki, and Kd in so101_mujoco_pid_utils.py.
